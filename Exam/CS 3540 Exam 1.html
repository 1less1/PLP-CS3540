Exercises

  1. Write brief answers for each of the following items (two sentences each, max).

    • One of the three things that every programming language has is a means of combination. What is Racket's only means of combination? Give an example to illustrate your answer.

    • What are the other two things that every programming language has? Give an example of each from Racket.

    • Explain the distinction in Racket between a function and a special form. Illustrate your answer with a specific example of each.

    • What does it mean to curry a function? What does a curried function look like in Racket?

  3. In the quadratic formula, the expression b� - 4ac is known as the discriminant. Write a Racket function named discriminant that takes three arguments, all numbers: the coefficients a, b, and c. Your function should return the value of the discriminant. For example: 
         > (discriminant 1  4 -5)     ; equation has two roots

  4. Write a Racket function named compare that takes two arguments, functions f and g, each of which takes a single integer argument.

    compare returns a function of one argument, an integer n. The returned function computes the value f(n) > g(n). For example: 
        > (define double (lambda (n) (* n 2)))
    > (define square (lambda (n) (* n n)))
    > (define 2n+1   (lambda (n) (add1 (double n))))

    > ((compare square double) 2)  ; is (2*2) > (2*2)?
    #f
    > ((compare square double) 5)  ; is (5*5) > (5*2)?
    #t

    > (define 2n+1>n�? (compare 2n+1 square))

    > (2n+1>n�? 2)     ; because 5 > 4
    #t
    > (2n+1>n�? 5)     ; because 11 is not > 25
    #f
    You'll earn bonus points if you don't use an if or a cond in your solution. Boolean values are values, too!

  5. Suppose that we have a list of pixels from an image, where each pixel is a list (r g b) containing the red, green, and blue (RGB) values of the pixel: 
         ( (76 195 202) (255 71 208) (175 81 214) (0 165 81) ... )
    Write a Racket function named max-average that takes one argument, a list of pixels lst in this form. max-average computes the average of the RGB values for every pixel in lst and returns the largest of the averages. For example: 
         > (max-average '((1 3 5)))
     3

     ; the averages are 120 and 240, respectively
     > (max-average '((100 120 140) (240 240 240)))
     240

     ; trust me:        this pixel |||||||||||| has the largest average
     ;                             vvvvvvvvvvvv
     > (max-average '((76 195 202) (255 71 208) (175 81 214) (0 165 81)))
     178
    Use higher-order functions apply and map to implement your function. No recursion or looping are allowed!

    You will want to use Racket's variable-arity max in your solution. You will also want to use the average function from class. It is variable-arity, too.
